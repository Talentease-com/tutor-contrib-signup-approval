signup-approval plugin for `Tutor <https://docs.tutor.edly.io>`__
#################################################################

Admin approval for signups


Installation
************

.. code-block:: bash

    pip install git+https://github.com/Talentease-com/tutor-contrib-signup-approval

Usage
*****

.. code-block:: bash

    tutor plugins enable signup-approval
